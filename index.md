# 首页

UniversalPluginUpdater，或简称 UPU，是一款强大的 Minecraft 服务端插件，允许您使用类似 APT 的命令快速检查现有插件的更新，并直接安装所有更新。

虽然它的主要作用是帮助服务器管理员保持服务端上的插件最新，但插件开发者也可以使用它提供的接口为自己的插件进行自动的更新检查，例如：
- 通过贡献插件更新配置让其他用户使用 `/upu repo get` 直接获取配置。
- 通过注册自己的 `AbstractUpdate` 实例来定制更新检查逻辑。

UPU 还有更多功能可以探索，同时也在不断增加新功能并不断完善已有功能。如果您是第一次使用 UPU，我们建议您从[安装指南](/setting-up)开始阅读。您可以在边栏中找到所有可用的帮助页面。
